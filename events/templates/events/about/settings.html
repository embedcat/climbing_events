Навзвание настройки Описание функции
Количество трасс Количество трасс, на которых соревнуются участники
Событие опубликовано Включает/отключает отображение события на главной странице и доступность для участников
Регистрация на событие разрешена Включает/отключает возможность регистрации участников
Просмотр результатов разрешён Включает/отключает просмотр страницы результатов
Ввод результатов разрешён Включает/отключает возможность ввода результатов участниками
Учитывать только введённые результаты Включает/отключает отображение участника в результатах и протоколах, а так же учёт участника без результата в подсчёте результатов по системе "От количества пролазов" (см. п. Тип подсчёта результатов)
Показывать полные результаты Включает/отключает отображение результатов (пролазов) участников на странице результатов
Показывать цвет трассы Включает/отключает отображение цвета трасс при введении результатов и на странице результатов. Цвет трасс настраивается на странице Трассы
Показывать категорию трассы Включает/отключает отображение категории трасс при введении результатов и на странице результатов. Категория трасс настраивается на странице Трассы
Показывать стоимость трассы Включает/отключает отображение стоимости трасс странице результатов
Рассчитывать стоимость трассы отдельно по каждой группе При выборе типа подсчёта результатов "От количества пролазов" и при наличии нескольких групп участников (например: Спортсмены, Новички):
Включено: стоимость трассы рассчитывается отдельно по каждой группе
Отключено: стоимость трассы рассчитывается по всем участникам одного пола, независимо от группы
Ввод результатов без регистрации Возможность вводить результаты без регистрации (и SMS :)), при этом предварительная регистрация участников невозможна. Данную возможность следует использовать при проведении мероприятий без предварительной регистрации.
Показывать участнику пин-код после регистрации Позволяет включить отображение страницы с пин-кодом участника после его регистрации.
При вводе результата показывать участнику страницу потверждения Позволяет включить отображение страницы проверки введенных результатов, где участник ещё раз сверяет свой введённый результат. Оттуда он может вернуться к редактированию результата или окончательно подтверждает свой результат.
Разрешить участнику обновлять свои результаты Позволяет участнику отредактировать свой результат по пин-коду. При выключенной настройке редактировать результат сможет только организатор события.
Тип подсчёта результатов Результат участника определяется суммированием всех баллов за трассы, которые он пролез, с учётом настроек Очки за трассу, Увеличение очков за Flash. Балл за трассу определяется:
Сумма баллов: числовым значением из настроек Очки за трассу ;
От количества пролазов: по формуле Балл = Б / Ку, где Б - базавая стоимость трассы (Очки за трассу с возможной надбавкой за Flash), Ку - количество участников одного пола, которые пролезли данную трассу (общее или только в определенной группе - см. п. Рассчитывать стоимость трассы отдельно по каждой группе).
Например, Очки за трассу - 80, надбавка за flash - 25%, Миша залез трассу на flash, а Илья на redpoint, тогда Миша получит (80*1,25)/2 = 50 баллов, а Илья (80)/2 = 40 баллов за эту трассу.
По таблице категорий: числовым значением из таблицы соответствия категории трассы и очкам за категорию - в настройках Трассы ;
По количеству Всего/Flash: общим количеством пройденных трасс и количеством трасс на flash;
Очки за трассу Базовая стоимость трассы (рекомендуемое значение - 80).
Увеличение баллов за Flash в % Коэффициент увеличения стоимости трассы при прохождении на flash (рекомендуемое значение - 25%).
Количество групп участников Количество различных групп участников, например по уровню лазания или по возрасту.
Список групп через запятую Список групп необходимо указать через запятую. Например: "Новички, Любители, Спортсмены" или "10-14 лет, 15-16 лет".
Количество сетов Количество сетов соревнований
Список сетов через запятую Список сетов необходимо указать через запятую. Например: "Сет 1 (10:00-12:00), Сет 2 (12:30-14:30)".
Максимальное число участников в сете (0 - не ограничено) Указать максимальное количество участников в каждом сете. По достижению этого лимита, регистрация в данный сет закрывается.
Дополнительные поля формы регистрации Выбирается, какие поля (помимо обязательных Имя, Фамилия) будут отображаться в форме регистрации участников. Список полей: Пол, Год рождения, Город, Команда, Разряд, Email.
Обязательные поля при регистрации Выбирается, какие поля в форме регистрации будут обязательными для заполнения.
Минимальный возраст участника Включает проверку на минимальный возраст участника при регистрации, 0 - проверка отключена.
